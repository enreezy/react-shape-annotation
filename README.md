# bbox-annotator

Online annotation tool created with ReactJS that can annotate objects in videos and images to get the height and width of the object use it for computer vision datas

## Installation
--------------------
Install Package dependencies

```
npm install
```

## Running the app
--------------------
```
npm run dev
```
you can now open <http://localhost:8080/> to your browser

<img src="https://media.giphy.com/media/Nlo1NSpzUiIVHRRirc/giphy.gif" width="100%"/>

## How to use

> Click the ADD NEW BUTTON to draw a box in a video, you must pause the video before you draw
> Move the box to follow the moving object. You need to resume and pause again before you move the box
